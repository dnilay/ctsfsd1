Exercise 1: Installation of Docker


In this exercise, you are going to install Docker.
  1. If your platform supports it, install Docker for native platform
  2. Otherwise, install Docker Toolbox
  3. Verify the installation

(1.1) If your platform supports it, install Docker for native platform

1. Download Docker package from https://docs.docker.com/engine/getstarted/step_one/
2. Install the package
3. If you see the following message, your machine is not equipped to install Docker natively.  Go to to the step (1.2) to install Docker Toolbox.
